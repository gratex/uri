# Convenient uri API


## Install
Node module, so use:

```
npm install @gjax/uri  
npm install @gjax/Uri  
```

## Usage

```
const uri = require('@gjax/uri');  
const res = uri.decomposeComponents('http://a/b/c/d;p?q');  
```
