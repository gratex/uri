# Convenient uri API


## Install
Node module, so use:

<code> 
npm install @gjax/uri<br>
npm install @gjax/Uri<br>
npm install @gjax/uri<br>
</code>

## Usage

<code> 
const uri = require('@gjax/uri'); <br>
const res = uri.decomposeComponents('http://a/b/c/d;p?q');
</code>
